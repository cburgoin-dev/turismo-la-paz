# 🌊 Turismo La Paz

Mobile application designed to help residents and visitors discover tourist attractions and experiences in **La Paz, Baja California Sur, Mexico**.

The app centralizes more than **40 tourist attractions**, providing location-aware recommendations, category-based exploration, and practical information to help users plan their visits more efficiently.

This project was presented before the **La Paz City Hall and the Municipal Tourism Department** as part of an academic collaboration initiative.

---

## 📱 Features

### 🗺️ Explore Attractions

Browse a curated collection of tourist destinations in La Paz, including:

- Beaches
- Viewpoints
- Cultural attractions
- Recreational areas
- Nature experiences
- Local points of interest

### 📍 Geolocation Support

- Detects the user's location
- Calculates distances to nearby attractions
- Displays recommendations based on proximity

### 🔎 Category Filtering

Users can quickly discover places using predefined categories such as:

- Relax
- Explore
- Near Me
- Budget

making navigation faster and more intuitive.

### 🧭 Attraction Details

Each attraction includes relevant information such as:

- Description
- Recommendations and tips
- Distance from the user
- Category information
- Direct access to navigation

### 🗺️ Google Maps Integration

Open attractions directly in Google Maps to obtain real-time directions.

### 🌐 Bilingual Experience

The application supports both:

- English
- Spanish

allowing it to serve both local users and international visitors.

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/cburgoin-dev/turismo-la-paz.git
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Run the App

Start the Expo development server:

```bash
npm start
```

Then open the application using:

- Expo Go, or
- an Android emulator.
